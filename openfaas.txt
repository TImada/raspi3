- Installation of OpenFaaS on Raspberry Pi 3 (aarch64) with Docker Swarm
1. Obtain the source code of OpenFaaS from the github Web site
-> git clone https://github.com/openfaas/faas
-> git checkout 0.8.0 (or higher version number)

2. Modify docker-compose.arm64.yml in the created directory
-> add the following line to the "gateway" section
     functions_provider_url: "http://faas-swarm:8080/"

3. Deploy a stack named as "func" in the directory
-> docker stack deploy func -c ./docker-compose.arm64.yml
